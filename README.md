# Shipping Route Mapper

A web application for visualizing and planning shipping routes on an interactive map.

## Technologies Used

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- React Leaflet (for map visualization)

## Run Locally

To clone and run this project locally, follow these simple steps:

```bash
# Clone the repository
git clone https://github.com/akhilchibber/shipping-route-mapper.git

# Navigate to the project directory
cd shipping-route-mapper

# Install dependencies
npm install

# Run the development server
npm run dev
```

The application will be available at `http://localhost:5173`
